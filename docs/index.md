# rclcpp_tutorials

ROS2の初学者向けチュートリアルです。

<br>

## 互換性

### ROS-Distro

Humbleを対象にしています。

ただし、GalacticやFoxyなどの過去バージョンに対しても通じることもあるので、その際は、その都度示している互換性の表を確認してください。

### OS

主にUbuntuを対象にしています。

<br>
